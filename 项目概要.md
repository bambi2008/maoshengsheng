## 一、一句话定位

**猫省省 — 给猫奴的AI省钱助手。拍照诊断 + 比价买药 + 智商税避坑。让猫过得好，不花冤枉钱。**

> 命名决策：2026.6.26 定名「猫省省」。叠字好记年轻化，"省"=省钱+省心。App Store格式：猫省省：AI养猫省钱助手

## 三、产品功能

```

│
├── 📸 AI症状拍照诊断（核心）
│   ├── 猫癣 / 异常掉毛 / 油毛黑下巴
│   ├── 呕吐物分析（毛球/食物/胆汁/血）

├── 🧠 智商税避坑
│   ├── 猫粮/保健品/用品测评数据库
│   ├── "这东西值不值"查询
│   └── 省钱攻略

├── 📋 健康档案
│   ├── 驱虫/疫苗提醒
│   ├── 就诊记录

│
└── 🔗 保险导流
    └── 众安/平安宠物险对比 → 投保（佣金80-160元/单）
```

## 四、盈利模型

| 收入线 | 单人年价值 | 说明 |
|--------|----------|------|
| 会员订阅 | 68元/年 | 无限次AI分诊+比价+避坑 |
| 药品CPS | 12-20元/年 | 京东联盟2-5%佣金 |
| 保险导流 | 80-160元/单 | 首年保费20-40%佣金 |
| 单次AI分诊 | 3.9元/次 | 覆盖不订阅用户 |

```
10万用户基准年收入：
  会员：5,250人 × 68元 = 35.7万
  CPS：30,000人 × 15元 = 4.5万
  保险：2,000单 × 100元 = 20万
  单次：10,000次 × 3.9元 = 3.9万
  ─────────────────────────────
  合计：~64万/年（单人运营净利30-40万）
```

## 五、核心差异化

```
宠智灵（最大竞品，600万用户）：
  → 做B2B（3万+宠物医院是客户）

  → 唯一做"诊断→比价→下单"闭环的
  → 利益和猫奴一致，不是和医院一致
  → 宠智灵不可能跟进（伤B端客户）
```

## 七、法律合规（关键）

| 红线 | 说明 |
|------|------|
| ❌ AI生成处方 | 《互联网诊疗监管细则》严禁 |
| ❌ 自营处方兽药 | 需《兽药经营许可证》 |
| ✅ AI辅助分诊 | 信息展示+建议 ≠ 诊断 |
| ✅ OTC导购（CPS） | 跳转京东/美团，不碰交易 |
| ✅ 免责声明 | "AI辅助工具，不替代兽医" |

**合规策略**：
- 不诊断（"你的猫得了X病"），做信息匹配（"常见于..."）
- 不处方（"用X药"），做方案展示（"常见选择包括..."）
- 不分轻重一律"建议就医"=自毁。要分层输出（🟢轻症给方案/🔴重症才去医）
- App Store分类选"生活"，不是"医疗"
